Для редакции непостоянных данных бота создан файл config.py

Перед тем как использовать бота НУЖНО:
1) Установить все нужные библиотеки, ввести команду pip install -r requirements.txt в директории с этим файлом (выполнять все в cmd (Командная строка))
(Для перемещения в директорию надо: скопировать путь и написать:
Cd {скопированный путь})
2) Добавить его в канал как подписчика
3) Выдать админ права
4) Поменять channel в config.py на нужно значение
5) Поменять admin в config.py на нужно значение

В случае ошибок ..."chat id not found..." Убедиться в правильности всех введенных данных
